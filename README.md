# SEYN — Landing Page

Brand landing page for the SEYN ecosystem at [www.seyn.co.uk](https://www.seyn.co.uk).

## Tech Stack

- React + Vite
- Deployed on Vercel

## Running Locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel dashboard
3. Point `www.seyn.co.uk` DNS to Vercel
